# React Demo - 📝 Notes App

This project is a classic notes tracking app.

> This was built following the Udemy Course: Modern React From The Beginning.
>
> _My Learning Philosophy: Build first, watch second._

## 🗝️ Key Learning Objectives

* Tailwind CSS

## 🔄 How to Run

1. Install dependencies: `npm install`
1. Run the app: `npm run dev`
1. Go to [http://localhost:5173/](http://localhost:5173/)
1. Add your notes!
